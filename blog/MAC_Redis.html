Mac环境下安装Redis

发表于 2019-10-26   |   分类于 技术篇

本文主要是mac环境下如何配置redis服务。

一、首先是官网下载redis

二、到下载目录找到下载好的压缩包并解压

1、移动到：mv redis-5.0.5 /usr/local/
2、切换到：cd /usr/local/redis-5.0.5/
3、编译测试：sudo make test
4、编译安装：sudo make install

安装成功测试

安装成功后，输入redis-server启动redis
